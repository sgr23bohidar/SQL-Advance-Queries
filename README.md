This repository contains foundational SQL queries and database assignments created to practice and demonstrate core concepts of relational databases and SQL operations.

It covers real-world use cases like:
Employee & Salary Management
Customer & Order Processing
University Database Management
Data retrieval, insertion, and updates

These exercises simulate how SQL is used in business systems and reporting environments.

**Repository Structure**
Folder / File	Description
Assignment_4_Select_Insert_Update	Practice of SELECT, INSERT, UPDATE, DELETE queries
Assignment_Employee Details and Salary	Employee and payroll database queries
Assignment_University Database Management System	University student, courses, and faculty data
Assignments5_Customers_Orders_Products Database	Customer order management system
SQL_Assignments_2 (EmployeeDetails_EmployeeSalary)	Relational queries on employee data
SQL_Assignments_3 (University_DBMS)	University DBMS queries
SQL_Assignments_4 (SELECT, Update, Insert Queries)	Core SQL operations
SQL_Assignment_5 (Customers_Orders_Projects_Database)	Business-oriented transactional database

**SQL Concepts Covered**

This repository focuses on core SQL fundamentals, including:
SELECT, INSERT, UPDATE, DELETE
WHERE, ORDER BY, GROUP BY
INNER JOIN, LEFT JOIN
Subqueries
Primary & Foreign Keys
Constraints
Relational database design

**Business Use Cases**

These queries are built around real-life business scenarios, such as:
Managing employee records and salaries
Tracking customers, orders, and products
Maintaining university student and course data
Generating simple business reports

**Objective**

The goal of this project is to build a strong SQL foundation for:

Business Analysts
Data Analysts
SQL Developers
Freshers preparing for interviews
